O Jogo de Caça-Palavras consiste de letras arranjadas aparentemente aleatórias em uma grade quadrada. O objetivo do jogo é encontrar as respostas das perguntas propostas. Essas respostas estão escondidas na grade de letras. veja a seguir as regras do jogo:

1. Em cada rodada haverá múltiplas pergunta.

2. Cabe ao jogador encontrar as respostas escondidas na grade.

3. Para cada resposta encontrada o jogador ganhará 100 pontos.

4. O jogador tem a opção de comprar uma dica por partida, nesse hipótese ele deverá pagar 25 pontos.

5. Se o jogador não encontrar todas as palavras escondidas na partida ele tem a opção de passar adiante.

6. Ganhará a partida o aluno que receber pelo menos 75% dos pontos.
